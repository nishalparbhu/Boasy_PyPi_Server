Boasy PyPi Server

This is the Boasy PyPi Server which is the Boasy Python package index to hold all our internal Python packages.

For more information, please see the repository README.md on how to use this index.

To pull a package from this index via the command line you need to enter the following:

pip install {package_name} --extra-index-url https://nishalparbhu.github.io/Boasy_PyPi_Server/

This file isn't used by pip, although it is still useful from a discoverability point of view though, and future versions of Pip may require it.

Pip will look for directories using the normalised name of your package ('python-hello' in this case instead of 'python_hello'). More details in the specification.
